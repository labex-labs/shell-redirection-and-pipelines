Linux 에서 쉘 리디렉션 및 파이프라인의 강력한 기능을 마스터하세요. 표준 입력, 출력 및 오류 스트림을 리디렉션하고, 파이프를 사용하여 명령을 연결하고, 명령 기록을 관리하며, sort 및 uniq 를 사용하여 텍스트 데이터를 처리하는 방법을 배웁니다. 이 모듈은 간단한 명령을 강력한 데이터 처리 워크플로로 결합하는 방법을 가르쳐줍니다.
